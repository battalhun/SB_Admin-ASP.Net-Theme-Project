# SB Admin ASP.NET Theme Project

Bu proje, [Start Bootstrap SB Admin](https://startbootstrap.com/template/sb-admin) HTML şablonunu temel alarak ASP.NET Core MVC ile entegre edilmiştir. Temayı kullanarak hızlıca dashboard ve yönetim panelleri oluşturabilirsiniz. Proje, dark/light tema duyarlılığı ile modern bir kullanıcı arayüzü sunar.

## Özellikler

- Dark ve light tema desteği
- Chart.js ile grafikler (Area Chart ve Bar Chart)
- Bootstrap 5 uyumlu responsive tasarım
- Simple-DataTables tablosu ile veri yönetimi
- ASP.NET Core MVC ile backend entegrasyonu
